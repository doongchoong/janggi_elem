빈 장기판 생성

<janggi-board></janggi-board>
<script src="js/janggi_board.js" ></script>

janggi-board 태그를 생성한 뒤 에 janggi_board.js 스크립트를 로드하면 장기판이 생성됩니다. 스크립트는 한번만 로드하면 됩니다.

시작위치 (Position)

<janggi-board position="start"></janggi-board>
<janggi-board position="[doubled-knight][coupled-knight]"></janggi-board>
<janggi-board position="3aka3/7re/4cc3/1ppppp3/r8/EN7/1PPPPP3/4CC3/9/3AKA3 w - - 0 1"></janggi-board>

빈 장기판에서 기물들을 시작 위치시킬수 있습니다.

  1. "start" : 초 좌귀마, 한 우귀마 기본값으로 기물들이 위치합니다.
  2. "[left-knight][right-knight]" : [한][초] 시작 포지션입니다.
    1. left-knight : 좌귀마
    2. right-knight : 우귀마
    3. doubled-knight : 양귀마
    4. coupled-knight : 원앙마
  3. FEN : 체스의 FEN 표기법을 사용할 수 있습니다.
    1. 대문자: 초나라, 소문자: 한나라
    2. K k (King) 왕
    3. R r (Rook) 차
    4. N n (kNight) 마
    5. E e (Elephant) 상 혹은 B b 도 가능
    6. A a (Advisor) 사
    7. C c (Cannon) 포
    8. P p (Pawn) 졸/병

시작 방향 (Orientation)

<janggi-board position="start"
orientation="han" ></janggi-board>
<script src="js/janggi_board.js" ></script>

기본적으로 초의 방향에서 장기판이 보이지만 한의 방향으로 변경할 수 있습니다.

이동가능 (Moveable)

<janggi-board position="start"
  moveable-piece ></janggi-board>
<script src="js/janggi_board.js" ></script>

기물을 이동 할 수 있는 모드입니다. moveable-piece 어트리뷰트를 설정하면 됩니다.

기본적으로 초/한 순서를 바꿔가며 기물을 움직일 수 있으며 기물이 움직일수 있는 위치로만 이동이 가능합니다.

클릭 하면 이동 가능한 위치가 표시됩니다. 이동 가능 위치를 클릭하면 이동합니다.

무규칙이동 (Ilegalmove)

<janggi-board position="start"
  moveable-piece
  ilegalmove ></janggi-board>
<script src="js/janggi_board.js" ></script>

기물이 무규칙으로 마음대로 이동할 수 있는 모드입니다. 이동가능 모드일때 작동됩니다.

체스기물 (Chess-pieces)

<janggi-board position="start"
  moveable-piece
  chess-pieces>
  </janggi-board>

기물 모양이 체스기물 모양입니다.
